🧠 Artificial Intelligence & Intelligent Agents

1. Fundamentals of Agents

  • Agent: Anything that perceives its environment through sensors and acts upon it through actuators.
  • Human Agent: Eyes/ears = sensors; hands/legs/vocal tract = actuators.
  • Robotic Agent: Cameras/infrared = sensors; motors = actuators.
  • Software Agent: Keystrokes/files = sensors; display output = actuators.
  • Perception: Passive interaction gaining info via sensors without changing the environment.
  • Action: Active interaction changing the environment via an "Effector".

2. Agent Behavior & Rationality

  • Intelligent Agent: Autonomous entity acting on an environment to achieve goals (e.g., Thermostat).
  • Rational Agent: Has clear preferences, models uncertainty, and maximizes performance measure.
  • Rationality vs Omniscience: Omniscient agents know actual outcomes (impossible in reality); rational agents maximize expected outcomes based on percept sequence.
  • Autonomy: System is autonomous if it acts according to its own experience rather than just built-in knowledge.

3. Nature of Environments

  • Fully vs Partially Observable: Can the sensor access the complete state at all times? (Unobservable = no sensors).
  • Deterministic vs Stochastic: Deterministic = current state + action completely determine next state. Stochastic = random nature.
  • Episodic vs Sequential: Episodic = one-shot actions. Sequential = requires memory of past actions.
  • Single-agent vs Multi-agent: Self-explanatory.
  • Discrete vs Continuous: Discrete = finite states/actions (Chess). Continuous = infinite (Self-driving car).
  • Static vs Dynamic: Dynamic = environment changes while agent is deliberating.

4. Agent Structure & Types

Agent = Architecture + Agent Program

  • Table Driven Agent: Looks up a pre-defined table. Drawback: Too big to store (Chess = 10^120 states), no non-perceptual knowledge, not adaptive.
  • Simple Reflex Agent: Decisions based ONLY on current percept (e.g., Vacuum: "If dirty then suck"). Fails in partially observable environments.
  • Model-based Reflex Agent: Maintains internal state based on percept history. Uses a "Model" of how the world works.
  • Goal-based Agent: Expands model-based by adding "goal" info. Uses Search and Planning.
  • Utility-based Agent: Adds utility measurement to choose the best way to achieve a goal among multiple alternatives.
  • Learning Agent: Adapts via experience. Components: Learning element, Critic, Performance element, Problem generator.

5. Defining Artificial Intelligence

  • AI: A branch of computer science enabling machines to simulate human cognitive functions (learning, reasoning).
  • Four Views of AI:
  • 1. Acting Humanly: Turing Test (requires NLP, Knowledge Rep, Reasoning, ML).
  • 2. Thinking Humanly: Cognitive science model (getting inside the human mind).
  • 3. Thinking Rationally: Laws of Thought / Logicist approach (Aristotle's syllogisms).
  • 4. Acting Rationally: The Rational Agent approach (achieving goals given beliefs).

6. Core Areas of AI

  • Machine Learning (ML): Learning from data to predict without direct programming.
  • Deep Learning (DL): Subfield of ML using artificial neural networks with many layers.
  • Natural Language Processing (NLP): Understanding and generating human language (Siri, chatbots).
  • Computer Vision: Interpreting visual info (facial recognition, self-driving cars).

7. Types of AI (By Capability & Function)

By Capability

  • Artificial Narrow Intelligence (ANI): Only form currently existing. Specific tasks (voice assistants, LLMs like Gemini).
  • Artificial General Intelligence (AGI): Theoretical. Broad tasks, human-like reasoning and adaptability.
  • Artificial Superintelligence (ASI): Theoretical. Self-aware, far surpassing human intelligence. Potential existential threat.

By Functionality

  • Reactive Machines: No memory, preprogrammed rules (IBM Deep Blue).
  • Limited Memory: Short-term memory for training (Self-driving cars, Chatbots).
  • Theory of Mind: Does not exist yet. Emulates human mind/emotions.

8. History & Milestones

  • 1943: McCulloch & Pitts propose artificial neurons.
  • 1950: Alan Turing publishes Turing Test.
  • 1956: John McCarthy coins "Artificial Intelligence" at Dartmouth Conference.
  • 1966: ELIZA (first chatbot).
  • 1974-1980 & 1987-1993: AI Winters (funding shortages).
  • 1980: Rise of Expert Systems.
  • 1997: IBM Deep Blue beats Gary Kasparov.
  • 2011: IBM Watson wins Jeopardy.
  • 2014: Chatbot "Eugene Goostman" passes Turing Test.
